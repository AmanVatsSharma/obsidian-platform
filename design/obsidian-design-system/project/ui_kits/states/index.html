04 / Decision matrix Which pattern to reach for.
Situation Pattern Dismissal Blocks UI?
Empty table / list <EmptyState> with CTA — (populates naturally) no
Initial data fetch Skeleton rows, matching shape Auto (data arrives) no
User-initiated submit Spinner on button, disable form Auto (response) yes · within scope
Background resync Pulse dots · top-bar indicator Auto no
Validation failure Inline ring · field-local message User corrects no
Rejected request (4xx) Inline error with next-action User acts or dismisses no
Throttled (429) Banner with countdown Auto (timer) no
Feed disconnect Top-bar red · full-state on trade attempt Reconnect success partial · trade blocked
Server error (5xx) Full-state · retry + status link Retry succeeds yes
